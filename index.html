Summary: Latitude vs X

The purpose of this project was to visualize the weather of 500 or more cities across the world and compare how the latitude of these cities affected different weather variables. These variables included temperature, humidity, cloudiness and wind speed.

We used the OpenWeatherMap API to assemble the dataset. After the data was assembled, we used Matplot lib to plot the comparisons of the weather variables versus latitude. This site provides the source data as well as the visualizations created as part of the analysis. On the individual plot pages you can find explanations of the data as well as descriptions of observed trends and correlations.
